Les opérateurs en JavaScript

Les opérateurs permettent d'effectuer des opérations sur des variables et des valeurs.

Exemples :

// Addition
let a = 5 + 3;

// Soustraction
let b = 10 - 4;

// Multiplication
let c = 6 * 2;

// Division
let d = 9 / 3;
