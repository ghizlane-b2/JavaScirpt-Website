Chaînes & Tableaux en JavaScript

Les chaînes de caractères et tableaux sont deux types essentiels en JavaScript.

Exemples :

// Chaîne
let text = "Hello World";

// Tableau
let fruits = ["Apple", "Banana", "Cherry"];
